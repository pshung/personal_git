---
name: No workarounds
description: Never workaround issues by changing existing data (e.g. shapes); always fix the root cause
type: feedback
---

Never workaround issues. Always fix the root cause instead of modifying existing data to compensate. For example, don't change input/output shapes to work around a template bug — fix the template itself.

Quick solution/fix is always the slowest solution/fix. Take the time to understand the problem fully and fix it properly.
